Build Safety Setup & Configuration

Step-by-step guide to deploy and configure the system

📋 Prerequisites

System Requirements
  • Java 17 or higher
  • MySQL 8.0+
  • Maven 3.6+
  • Git for version control
  • Modern web browser
Required API Keys
  • Google Maps API Key
  • OpenWeather API Key
  • JWT Secret Key
  • Database Credentials

Step 1

Database Setup

🗄️ MySQL Database Configuration

Create and configure your MySQL database with the provided schema.

Commands
-- Create database
CREATE DATABASE smart_tourist_safety;
USE smart_tourist_safety;

-- Import schema
SOURCE schema.sql;

-- Verify installation
SHOW TABLES;

Step 2

Environment Setup

🔧 Configure Application Properties

Set up environment variables and application properties.

application.properties
# Database Configuration
spring.datasource.url=jdbc:mysql://localhost:3306/smart_tourist_safety
spring.datasource.username=root
spring.datasource.password=your_password

# API Keys
google.maps.api.key=YOUR_GOOGLE_MAPS_KEY
openweather.api.key=YOUR_OPENWEATHER_KEY

# JWT Configuration
jwt.secret.key=your_jwt_secret_key_here
jwt.expiration=86400000

# Server Configuration
server.port=8080
server.servlet.context-path=/api

Step 3

Build & Deploy

🚀 Build and Run the Application

Compile and run the Spring Boot application.

Maven Commands
# Clean and build
mvn clean install

# Run the application
mvn spring-boot:run

# Or use JAR file
java -jar target/smart-tourist-safety-monitoring-system.jar

Step 4

API Integration

🌐 External API Integration

Connect with Google Maps and OpenWeather APIs.

Google Maps API
  1. Create Google Cloud account
  2. Enable Maps JavaScript API
  3. Create API key
  4. Restrict to your domain
  5. Add to application.properties
OpenWeather API
  1. Sign up on openweathermap.org
  2. Generate API key
  3. Verify key works
  4. Add to application.properties
  5. Test weather endpoints

✅ Verification Checklist

Backend Status
  • Spring Boot running on port 8080
  • MySQL database connected
  • All tables created
  • API endpoints responding
API Verification
  • Google Maps API working
  • OpenWeather API responding
  • JWT authentication functional
  • WebSocket connections established
